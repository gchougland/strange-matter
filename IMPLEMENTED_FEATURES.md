# Strange Matter Mod - Implemented Features List

This document provides a comprehensive list of all currently implemented features in the Strange Matter mod.

## üåå Anomaly System

### Anomaly Types (6 Types)
1. **Gravity Anomaly** (`GravityAnomalyEntity`)
   - Creates levitation fields
   - Modifies gravity for entities
   - Configurable levitation radius and force

2. **Temporal Bloom** (`TemporalBloomEntity`)
   - Accelerates crop growth
   - Transforms mobs over time
   - Time dilation effects

3. **Warp Gate** (`WarpGateAnomalyEntity`)
   - Enables teleportation between gates
   - Chunk loading system for gate networks
   - Mini warp gates for Warp Gun

4. **Energetic Rift** (`EnergeticRiftEntity`)
   - Generates electrical effects
   - Strikes lightning rods
   - Powers Rift Stabilizers

5. **Echoing Shadow** (`EchoingShadowEntity`)
   - Absorbs light
   - Boosts mob spawning rates
   - Shadow manipulation effects

6. **Thoughtwell** (`ThoughtwellEntity`)
   - Affects player perception
   - Confuses nearby mobs
   - Cognitive disguise system

### Anomaly Features
- Natural spawning in various biomes
- Configurable spawn rates per dimension
- Research point rewards when scanned
- Containment system via Echo Vacuum
- Anomalous grass generation around anomalies
- Resonite ore spawning near anomalies

## üéÆ Commands

### Anomaly Commands (OP Level 2)
- `/anomaly spawn <type> [pos]` - Spawn an anomaly
- `/anomaly locate <type>` - Find nearest anomaly of type
- `/anomaly list` - List all available anomaly types
- `/anomaly debug_chunk <pos>` - Debug chunk loading

### Research Commands (OP Level 2)
- `/research check [target]` - Check research progress
- `/research add <type> <amount> [target]` - Add research points
- `/research reset [target]` - Reset research progress
- `/research give <research_id> [target]` - Give research notes
- `/research unlock <research_id> [target]` - Unlock specific research
- `/research unlock_all [target]` - Unlock all research

### Research Points Commands
- `/research_points check [target]` - Check research points
- `/research_points` - Conditional command integration
  - Execute if integration
  - Scoreboard integration
  - Command block ready
